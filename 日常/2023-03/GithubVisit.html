Github

众所周知，因为某些原因 Github 偶尔就会无法访问，处理方法在网上找了好多次了，这次自己记录一下以备后续出问题了能用到。

主要的办法是手动修改 hosts 文件

手动修改hosts

step1

首先，需要手动查找到 GitHub 的 ip 地址，可以在下面的网站进行查询

  • https://www.ipaddress.com/site/github.com

这里我的查询结果如下：

即 140.82.114.3

Step2

得到 ip 地址后手动修改 hosts 文件，hosts 文件的路径为：

  • Windows 系统：C:\Windows\System32\drivers\etc\hosts
  • Linux 系统：/etc/hosts
  • Mac（苹果电脑）系统：/etc/hosts
  • Android（安卓）系统：/system/etc/hosts
  • iPhone（iOS）系统：/etc/hosts
